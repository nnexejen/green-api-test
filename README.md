# GREEN-API Client 

Веб-страница для прямого взаимодействия с API WhatsApp через сервис [GREEN-API](https://green-api.com/). Реализована на чистом HTML, CSS и JavaScript без использования серверной части.   
Страница находится по адресу: <https://nnexejen.github.io/green-api-test/>

## Используемые методы
Страница поддерживает вызов следующих методов API:

* ```getSettings``` - получение настроек текущего инстанса (GET)
* ```getStateInstance``` - проверка состояния подключения и авторизации (GET)
* ```sendMessage``` - отправка текстового сообщения в чат (POST)
* ```sendFileByUrl``` - отправка файла по прямой ссылке (POST)
️
## Реализация
**Технологии:** ```HTML5```, ```CSS3```, ```JavaScript```.
**Взаимодействие с API:** Используется встроенный в браузер Fetch API. Запросы отправляются напрямую с клиента на сервер GREEN-API.
**Безопасность:** Учётные данные (idInstance и apiToken) нигде не сохраняются и передаются только в рамках текущего сеанса браузера.
**Обработка данных:**
* Автоматическое добавление суффикса @c.us к номерам телефонов, если он не указан.
* Автоматическое извлечение имени файла из URL при отправке файлов.
* Форматированный вывод JSON-ответов в поле вывода.  

**Обработка ошибок:** Реализована проверка HTTP-статусов, валидация полей ввода и сообщения при сетевых или логических ошибках.

## Использование сервиса
### Начало работы
1. Зарегистрируйтесь в [личном кабинете GREEN-API](https://console.green-api.com/instanceList) и создайте инстанс для WhatsApp.
2. Перейдите на сайт <https://nnexejen.github.io/green-api-test/>
3. В поля ```idInstance``` и ```ApiTokenInstance``` введите соответствующие параметры из личного кабинета GREEN-API.
4. Убедитесь, что статус инстанса в методе getStateInstance отображается как authorized.
### Проверка настроек и состояния
1. Нажмите кнопку ```getSettings``` для получения конфигурации инстанса.
2. Нажмите кнопку ```getStateInstance``` для проверки статуса подключения.
Результат отобразится в правом блоке в формате JSON.
### Отправка сообщения
1. В поле номера телефона укажите номер получателя (например, 79991234567).
2. В поле текста введите сообщение.
3. Нажмите кнопку ```sendMessage```.   
В случае успеха API вернёт idMessage.
### Отправка файла по ссылке
1. В поле номера укажите получателя.
2. В поле URL вставьте прямую ссылку на файл (например, изображение или документ).
3. Нажмите кнопку ```sendFileByUrl```. Имя файла и подпись определяются автоматически.
